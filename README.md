# 裂变原理

裂变原理：
用户A点外卖，佣金的30%的返利给用户。
用户A邀请用户B，用户B点外卖，用户A作为用户B的上级，给佣金的10%作为奖励。
用户B邀请用户C，用户C点外卖，用户B作为用户C的上级，拿佣金的10%， 用户A拿佣金的5%，用户A实现了躺赚。
